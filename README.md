# SimpleClockJs
 A simple page that will display your local time
[View this app](https://simple-clock-js.herokuapp.com/)

## Built with

 - Vanilla Javascript
 - IP Geolocation API by IPify

## Useful resources

 - [IP Geolocation API by IPify](https://geo.ipify.org/)
